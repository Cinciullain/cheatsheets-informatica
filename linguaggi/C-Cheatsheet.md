### Operatori

**Operatori Aritmetici**

| Operatori | Esempio   | Significato        |
| --------- | --------- | ------------------ |
| +         | a + b     | Addizione          |
| ++        | a++ o ++a | Incrementa di 1    |
| -         | a - b     | Sottrazione        |
| --        | a-- o --a | Decrementa di 1    |
| *         | a * b     | Moltiplicazione    |
| /         | a / b     | Divisione          |
| %         | a % b     | Modulo             |

**Operatori Logici e Relazionali**

| Operatori | Esempio   | Significato        |
| --------- | --------- | ------------------ |
| <         | a < b     | Minore di          |
| <=        | a <= b    | Minore o uguale    |
| >         | a > b     | Maggiore di        |
| >=        | a >= b    | Maggiore o uguale  |
| &&        | a && b    | AND logico         |
| &#124;&#124; | a &#124;&#124; b | OR logico  |
| !            | a ! b o !a       | NOT logico |

**Operatori Bit a Bit**

| Operatori | Esempio   | Significato        |
| --------- | --------- | ------------------ |
| << |  | Shift a sinistra |
| >> |  | Shift a destra   |
| & | a & b | AND |
| ^ | a ^ b | XOR |
| &#124; | a &#124; b | OR | ~
| ~ | ~ (a) | C1 |
